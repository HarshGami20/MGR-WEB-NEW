# Blue Schema Manager

Two **independent** applications (no shared `lib/`, no monorepo workspace, no Replit config):

| App | Path | Stack |
|-----|------|--------|
| **Backend** | `backend/api-server` | Express, Prisma, PostgreSQL |
| **Frontend** | `frontend/furniture-erp` | React, Vite, React Query |

Install and run each from its own folder (see the `README.md` inside each app).

Optional: `frontend/mockup-sandbox` is a separate Vite app for UI experiments; it is not required for the ERP.
